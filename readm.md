# LeetCode Company-Wise Master Dataset (1,446 Questions)

A curated dataset of **1,446 unique LeetCode problems** aggregated from the community-maintained **LeetCode-Questions-CompanyWise** repository.

The dataset has been deduplicated, organized by topic, annotated with company tags, and sorted by difficulty to provide a structured resource for interview preparation.

## Files

### `leetcode_1446_company_topicwise.xlsx`

- 1,446 unique questions

- Separate sheets for each topic

- Company tags for every problem

- Number of companies in which each problem appears

- Difficulty information

### `leetcode_1446_company_topicwise_sorted.xlsx`

- Same dataset as above

- Questions within each topic are sorted by:

  1. Easy

  2. Medium

  3. Hard

- Further sorted by LeetCode problem ID

### CSV Versions

- `leetcode_1446_company_topicwise.csv`

- `leetcode_1446_company_topicwise_sorted.csv`

## Dataset Columns

| Column | Description |

|----------|-------------|

| Question ID | Official LeetCode problem ID |

| Title | Problem title |

| Difficulty | Easy / Medium / Hard |

| Topic | Topic classification |

| Companies | Companies where the question appears |

| Company Count | Number of companies that asked the question |

## Features

- Topic-wise organization

- Difficulty-wise ordering (Easy → Medium → Hard)

- Company tags for every question

- Company frequency counts

- Excel and CSV formats

- Deduplicated question set

## Intended Use

This repository can be used for:

- Software engineering interview preparation

- Company-specific question analysis

- Building structured study plans

- Tracking progress across topics

- Identifying high-frequency interview problems

## Notes

- The original data is sourced from the community-maintained **LeetCode-Questions-CompanyWise** repository.

- Duplicate questions across companies have been merged.

- Company tags indicate every company in which a question appears.

- Topic labels are generated heuristically and may not exactly match official LeetCode tags.
